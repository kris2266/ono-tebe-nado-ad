# Проект "Оно тебе надо"

Аукцион вещей, в которые никто не верил.

## Ссылка на репозиторий

https://github.com/kris2266/ono-tebe-nado-ad

## Технологии

- HTML5
- CSS3
- Flexbox
- Grid Layout

## Описание

Одностраничный сайт для аукциона необычных вещей. 
Вёрстка выполнена по макету из Figma с использованием семантической разметки и современных методов позиционирования.

## Структура проекта

- `index.html` — главная страница
- `styles/` — стили проекта
- `fonts/` — шрифты
- `images/` — изображения
